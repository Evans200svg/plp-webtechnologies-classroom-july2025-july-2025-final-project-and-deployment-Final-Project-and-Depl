Course Overview

In this course, you'll learn how to write Python code and build basic programs. It is designed for beginners who have no prior programming experience.

Syllabus:

  • Introduction to Python
  • Variables and Data Types
  • Control Flow (if, elif, else)
  • Loops (for, while)
  • Functions and Modules
  • Working with Libraries
  • Project: Build a Python Application

Prerequisites

No prior experience needed. A basic understanding of computers will be helpful.
